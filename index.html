👨‍🏫2022/03/01 class 02 自製電聯車和自由落體

發表於 2022-03-01 更新於 2022-04-09 分類於 class


本次社課是自製電聯車及自由落體，自製電聯車利用簡單、容易取得的銅線、電池以及強力磁鐵就能做出能自行行駛的電聯車！如果把線圈捲得更長，就能向小火車一樣持續的跑跑跑！

至於自由落體實驗該不會真的直接將東西丟下樓吧？！
是的
你也認為自由落體簡單到不行嗎？那麼以下的內容可要好好地讀
七組的社員，分別將鋼珠由四樓的高度，靜止下墜。 測得鋼珠自施放到落地的時間，並且取得樓高之數據。
依據樓高，並假設重力加速度為9.807 m/s，我們可以計算出下墜的時間。
兩相比較，竟發現實際測得下墜的時間較理論的時間短！
為什麼呢？
以下為我們的推論
一、鄰近建築物的氣流具有垂直移動的可能性。
二、社員的測量過程具有誤差，或是在施放時並非靜止落下。
（社幹們一致認為主要的原因為後者）
本次的社課安排是為了銜接本社的成果發表－探究競賽。傳達即使是看似簡單的主題在實際操作後，皆會發現有趣的現象或可以改進的操作方式，將實驗做到精確、準確的核心理念。
接下來的社課，將如火如荼地開始準備成果發表，希望各組皆能有好結果。

👨‍🏫class 3 10/06 化學花園

發表於 2021-10-27 分類於 class

本週社課的主題是：化學花園
本實驗的主角是-矽酸鈉。矽酸鈉融液與二、三價金屬鹽類相遇，會因為金屬替換，連結模式較一價鈉金屬更為複雜，形成膠狀沉澱。 以金屬鹽固體為種核形成透水半透膜，再透過內外濃度梯度差滲透漲破半透膜，結晶就會如同植物一般生長

活動一、觀察不同金屬的結晶樣貌
將金屬鹽放入觀察瓶底，加入矽酸鈉，靜置10~20分鐘，會看到小芽以無法察覺的速度緩慢成長。不同的矽酸化金屬有不同的晶形，因而生長的形狀顏色各不相同。

觀察到：硫酸鐵的生長較快、硫酸銅生長的慢而直，顏色是藍色、硫酸鋅自然產生的結晶能夠長的像海帶，顏色是白色等等…
觀察瓶就各自帶回家好好玩賞一番囉！
